# Siarter
Siarter (Charter in Welsh) is a TUI tool that uses the AIS API to visualise boats in the terminal.

# Installation

```sh
git clone https://github.com/rhydianjenkins/siarter
cd siarter
go install # this will place a binary in your GOPATH
```
